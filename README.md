# appzh-hth-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常用于实验、演示、工具或临时展示，不针对任何特定域名或网站。

仓库内容以静态 HTML 文件为主，部分可能附带 CSS、JavaScript 等资源文件。所有页面均为独立运行，彼此之间无依赖关系。

## 目录说明

```
├── pages/          # 存放各独立 HTML 页面及其附属资源
├── assets/         # 公共资源文件（如图片、样式、脚本）
├── docs/           # 项目文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹代表一个独立页面，文件夹名称即为页面标识。
- `assets/` 目录存放多个页面可能共用的资源，非必需。
- `docs/` 目录用于存放与项目相关的额外文档，非必需。

## 页面归档说明

每个独立页面均以文件夹形式归档，内部包含：

- `index.html`：主页面文件
- 其他资源文件（如 CSS、JS、图片等）

页面之间互不干扰，可单独部署或访问。

## 使用方式

1. 克隆或下载本仓库到本地
2. 在浏览器中直接打开 `pages/` 下对应页面的 `index.html` 文件即可查看
3. 也可将整个 `pages/` 目录部署到任意静态服务器上

## 维护说明

- 本仓库为归档性质，内容更新频率不定
- 如需新增页面，请在 `pages/` 下新建文件夹，放入完整页面文件
- 请保持文件结构清晰，避免与已有页面冲突
- 不建议在页面中加入外部跟踪、统计或营销代码

## 许可

本仓库内容仅供学习、实验和归档使用。如涉及第三方资源，请遵守其各自许可协议。
